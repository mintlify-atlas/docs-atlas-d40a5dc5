## Your first reading in four steps

<Steps>
  <Step title="Open Claude Code and trigger the skill">
    Start a Claude Code session in any project where Bazi Skill is installed. Type any of the following trigger phrases to begin:

    ```
    算八字  看八字  批八字  排八字  四柱  命盘  算命  排盘  bazi
    ```

    You do not need to prefix the phrase with a command or slash. As soon as Claude Code sees a trigger keyword, it activates the skill automatically.

    <Tip>
      Even a casual mention of "算命" or "八字" is enough to trigger the skill — you do not need to say "use the bazi skill."
    </Tip>
  </Step>

  <Step title="Answer the prompts one by one">
    The skill collects your birth information through a guided conversation. It asks for one piece of information at a time — never all at once:

    | Prompt | What to provide |
    |--------|----------------|
    | Name (姓名) | Your current legal name |
    | Former name (曾用名) | Any name you used before, or "no" to skip |
    | Solar birth date (阳历生日) | Gregorian date, e.g. `1990年5月15日` |
    | Lunar birth date (农历生日) | Lunar calendar date; note intercalary months (闰月) |
    | Birth hour (出生时辰) | Exact time or approximate period (e.g. "morning") |
    | Gender (性别) | Male (男) or Female (女) — affects Major Luck Cycle direction |
    | Birthplace (出生地) | Province and city, e.g. `辽宁省丹东市` |
    | Living status (在世状态) | Living or deceased (provide year if deceased) |

    <Note>
      If you do not know your lunar birth date, type "不知道". If you know neither your solar nor lunar date, the skill cannot generate a chart and will advise you to check your birth certificate.
    </Note>

    A sample exchange looks like this:

    ```
    Skill:  请告诉我您的姓名。
    You:    李明

    Skill:  请问您有曾用名吗？
    You:    没有

    Skill:  请告诉我您的阳历（公历）生日，格式如：1990年5月15日。
    You:    1988年3月22日

    Skill:  请告诉我您的农历（阴历）生日…
    You:    1988年二月初五

    Skill:  请问您的出生时辰是？
    You:    早上七点左右
    ```
  </Step>

  <Step title="Confirm the summary">
    After collecting all information, the skill displays a confirmation summary before calculating anything:

    ```
    📋 信息确认：
    - 姓名：李明（曾用名：无）
    - 阳历生日：1988年3月22日
    - 农历生日：1988年二月初五
    - 出生时辰：辰时（07:00–09:00）
    - 性别：男
    - 出生地：辽宁省丹东市
    - 当前日期：2026年4月8日
    - 在世状态：在世

    以上信息是否正确？如需修改请告诉我，确认后我将开始排盘分析。
    ```

    Review each field. If anything is wrong, tell the skill which field to correct. Once you confirm, it proceeds to the charting phase.
  </Step>

  <Step title="Receive your chart and analysis">
    The skill calculates your Four Pillars chart and delivers a structured analysis. Here is what the output covers:

    **Four Pillars table (四柱)**

    | | Year pillar (年柱) | Month pillar (月柱) | Day pillar (日柱) | Hour pillar (时柱) |
    |-|-------------------|---------------------|-------------------|-------------------|
    | Heavenly Stem (天干) | 戊 | 甲 | 丙 | 壬 |
    | Earthly Branch (地支) | 辰 | 寅 | 戌 | 辰 |
    | Ten God (十神) | 食神 | 偏印 | — | 七杀 |
    | Hidden Stems (藏干) | 戊乙癸 | 甲丙戊 | 辛丁戊 | 戊乙癸 |

    **Major Luck Cycle table (大运)**

    | Cycle | Age range | Stems & Branch |
    |-------|-----------|---------------|
    | Pre-onset | 0–7 | 甲寅 (monthly pillar) |
    | First | 8–17 | 乙卯 |
    | Second | 18–27 | 丙辰 |
    | Third | 28–37 | 丁巳 |

    **Analysis sections**

    - Day-master strength (日主旺衰) — with reference to 《滴天髓》
    - Ten Gods interpretation (十神分析) — with reference to 《渊海子平》
    - Five-Element balance and favorable/unfavorable elements (喜忌神) — with reference to 《穷通宝典》
    - Pattern determination (格局判定) — with reference to 《子平真诠》
    - Current Major Luck Cycle and annual luck (大运流年)
    - Historical calibration — the skill proposes 3–5 past events for you to confirm or correct
    - Life-area guidance covering career, relationships, health, and auspicious directions

    <Tip>
      Engage with the historical calibration step. The more accurately you confirm or correct the past-event predictions, the more precisely the skill can refine its model of your chart before looking ahead.
    </Tip>
  </Step>
</Steps>
